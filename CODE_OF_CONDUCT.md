# Code of Conduct

## Our Pledge

We are committed to making participation in Veridra Core a respectful, harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Using welcoming and inclusive language
- Being respectful of differing viewpoints
- Accepting constructive feedback gracefully
- Prioritizing community and project health

Examples of unacceptable behavior:

- Harassment, personal attacks, or discriminatory language
- Trolling, insults, or derogatory comments
- Publishing private information without permission
- Any conduct that is inappropriate in a professional setting

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.

## Scope

This Code of Conduct applies in all project spaces, including issues, pull requests, discussions, and other official channels.
