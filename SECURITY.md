# Security Policy

## Supported Versions

Security fixes are provided for the latest release on `main`.

## Reporting a Vulnerability

Please report suspected vulnerabilities privately:

- Email: `apusharma45@gmail.com`
- Subject: `SECURITY: veridra-core`

Include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any proof-of-concept details

Please do not open public GitHub issues for undisclosed vulnerabilities.

## Response Expectations

- Initial response target: within 5 business days
- Triage and severity assessment: as soon as reproducibility is confirmed
- Fix timeline: depends on severity and complexity
